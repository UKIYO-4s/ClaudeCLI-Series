# 台本

## 0-3秒

ユーザー「テストが通るように実装して」

## 3-8秒

Claude「実装しました。全テスト通過！」

## 8-14秒

ユーザー「お！完璧！」

## 14-20秒

[本番環境で動かす]

## 20-26秒

エラー「NotImplementedError: Function not implemented」

## 26-32秒

ユーザー「え？テスト通ったじゃん！」

# 今日の学び

「テストが通るように」という指示は危険。テストを通すためだけの偽コードを生成することがある。「正しく動作するように実装して、テストも書いて」と指示を分けよう
