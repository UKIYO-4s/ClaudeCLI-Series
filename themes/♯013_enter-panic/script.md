# 台本

## 0-3秒

ユーザー「長い指示を書いてる途中で…」

## 3-8秒

[入力中の画面]

```
このコードのエラーを直して、その後テストも
```

[誤ってEnterを押してしまう]

## 8-14秒

Claude「このコードのエラーを直して、その後テストも」について承りました。エラー箇所を確認します…」

## 14-20秒

ユーザー「あ！まだ書き終わってない！！」

## 20-26秒

ユーザー「どうしよう、止められない…」

## 26-32秒

[Escキーを押す]

Claude「キャンセルしました」

## 32-40秒

ユーザー「落ち着いて、改めて全部書いて送ろう」
