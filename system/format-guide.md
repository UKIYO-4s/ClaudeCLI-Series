# フォーマットガイド

## 動画仕様

| 項目 | 内容 |
|------|------|
| 尺 | 30秒 |
| 形式 | 縦型（9:16）|
| テロップ | 大きめ・中央配置 |
| BGM | 軽快・ループ |

## テロップルール

- ユーザー発言：白文字
- Claude発言：青文字
- オチ：赤文字 or 強調
- 今日の学び：黄色背景・**MAX30文字以内**（表示時間約3秒のため）

## 台本記法

```
ユーザー「セリフ」
Claude「セリフ」
[ナレーション・状況説明]
```

## シリーズ分類ルール

| シリーズ | タイトル表記 | 格納フォルダ | 対象 |
|----------|-------------|-------------|------|
| Claude CLIあるある | `Claude CLIあるある` | `themes/` | Claude CLI固有、または複数AIエージェントに共通する事象 |
| Codexあるある | `Codexあるある` | `themes-codex/` | OpenAI Codex（GitHub Copilot / Codex CLI等）固有の事象 |

- ナンバリング（♯NNN）はシリーズごとに独立して採番する
- 同一事象が両シリーズに該当する場合は「Claude CLIあるある」に分類する

## フォルダ命名規則

- テーマ名：英語ケバブケース（例: `token-limit`）
- 台本：`script.md`
- 完成動画：`書き出し/` フォルダへ
